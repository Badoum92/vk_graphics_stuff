# Graphics stuff using Vulkan

Learning graphics programming with Vulkan

## Build

Tested on Windows 10 only
```
$ git clone --recursive
$ cd vk_graphics_stuff
$ mkdir build && cd build
$ cmake .. -G Ninja
$ ninja
```
